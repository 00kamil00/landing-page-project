# Landing Page – The Odin Project

A simple landing page project built as part of the *Foundations* path on **The Odin Project**.

## 🎯 Goal
The main objective was to practice and master **CSS Flexbox** layout and replicate a provided design from scratch.

## 🛠️ Tech Stack
* HTML5
* CSS3 (Flexbox)

## 💡 Key Takeaways
* Positioning elements effectively using `display: flex`, `justify-content`, and `align-items`.
* Structuring semantic HTML (Header, Hero, Flex-grid, Quote, CTA, Footer).
* Managing consistent spacing with margins and paddings.
